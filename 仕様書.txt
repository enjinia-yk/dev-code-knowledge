■ 画面構成
------------------------------------------------------------

┌─────────────────────────────────────────────────────┐
│ [検索バー]                          [+ 新規登録]    │  ← 上部バー
├──────────┬──────────────────┬────────────────────────┤
│ 言語      │ スニペット一覧    │ コード詳細             │
│ タグ      │                  │                        │
└──────────┴──────────────────┴────────────────────────┘
  左ペイン    中央ペイン          右ペイン

------------------------------------------------------------
■ 検索機能
------------------------------------------------------------

【検索バー（上部）】

  検索対象:
    - タイトル
    - 説明
    - コード本文
    - タグ名

  仕様:
    - 文字を入力するたびにリアルタイムで絞り込まれる
    - 大文字・小文字を区別しない（例: "select" で "SELECT" もヒット）
    - 部分一致で検索される（例: "顧客" で "顧客一覧取得" がヒット）
    - 複数の検索対象を同時に見るため、タイトルになくてもコード内に
      あればヒットする
    - 入力欄の右端の × ボタンで検索をクリアできる

------------------------------------------------------------
■ 言語フィルター（左ペイン・上段）
------------------------------------------------------------

  仕様:
    - 登録済みの言語が一覧表示される
    - クリックするとその言語のスニペットだけに絞り込まれる
    - 選択中の言語はハイライト表示される
    - 「すべて」をクリックすると絞り込みが解除される
    - 言語フィルターと検索バーは同時に使える（AND条件）
    - セクションヘッダーをクリックすると折りたたみ・展開できる

  初期登録されている言語:
    SQL / C# / VBA / Python / JavaScript / PowerShell / その他

------------------------------------------------------------
■ タグフィルター（左ペイン・下段）
------------------------------------------------------------

  仕様:
    - スニペットに付けられたタグが自動で一覧に表示される
    - クリックするとそのタグを持つスニペットだけに絞り込まれる
    - 複数のタグを同時に選択できる
    - 同じタグをもう一度クリックすると選択が解除される
    - AND / OR 切替ボタンで絞り込み条件を変更できる
        OR: いずれかのタグを持つスニペットを表示（デフォルト）
        AND: すべてのタグを持つスニペットのみ表示
    - リセットボタン（×）で選択中のタグをすべて解除できる
      （タグが1つ以上選択されているときのみ表示）
    - タグフィルター・言語フィルター・検索バーは同時に使える（AND条件）
    - 使用されていないタグは自動で削除される
    - セクションヘッダーをクリックすると折りたたみ・展開できる

------------------------------------------------------------
■ ピン留めフィルター（左ペイン・最上段）
------------------------------------------------------------

  仕様:
    - 「📌 ピン留めのみ」ボタンをクリックするとピン留め済みの
      スニペットだけに絞り込まれる
    - 他のフィルター（言語・タグ・検索）と組み合わせて使える

------------------------------------------------------------
■ スニペット一覧（中央ペイン）
------------------------------------------------------------

  表示内容（1件につき）:
    - タイトル
    - ピンアイコン（📌）※ピン留め中のみ表示
    - 言語バッジ（青色）
    - タグバッジ（緑色）
    - 最終更新日

  並び順:
    - ピン留めスニペットが常に上部に表示される
    - ピン留め内・通常スニペット内の順序は以下から選択できる:
        更新日時（新しい順）/ 更新日時（古い順）
        作成日時（新しい順）/ 作成日時（古い順）
        タイトル（昇順）    / タイトル（降順）

  操作:
    - クリックすると右ペインにコード詳細が表示される
    - 選択中のスニペットは左端に青い線が表示される

------------------------------------------------------------
■ コード詳細（右ペイン）
------------------------------------------------------------

  表示内容:
    - タイトル
    - 説明
    - 言語バッジ・タグバッジ
    - 最終更新日時（時刻まで表示）
    - コード本文（シンタックスハイライト付き）

  シンタックスハイライト対応言語:
    SQL / C# / VBA / Python / JavaScript / PowerShell

  操作ボタン:
    [コピー]      コード本文をクリップボードにコピーする
                  → コピー成功時に「コピーしました！」と通知が出る
    [ピン留め]    ピン留めのON/OFFを切り替える
                  → ピン留め中はボタンが青くハイライトされる
    [複製]        同じ内容のスニペットを新規作成する
                  → タイトルに「のコピー」が付く
                  → ピン留め状態は引き継がれない
    [編集]        編集フォームを開く
    [削除]        確認ダイアログを表示してから削除する

  スニペット未選択時:
    「スニペットを選択してください」と表示される

------------------------------------------------------------
■ スニペット登録・編集フォーム
------------------------------------------------------------

  開き方:
    - 上部の「+ 新規登録」ボタン → 空の登録フォーム
    - 右ペインの「編集」ボタン   → 既存データが入力済みのフォーム

  入力項目:
    ┌───────────┬────────┬─────────────────────────────────┐
    │ 項目       │ 必須   │ 説明                             │
    ├───────────┼────────┼─────────────────────────────────┤
    │ タイトル   │ 必須   │ スニペットの名前                  │
    │ 説明       │ 任意   │ メモ・用途など                    │
    │ 言語       │ 必須   │ 選択または新規入力可能            │
    │ タグ       │ 任意   │ 選択または新規入力で複数付与可能  │
    │ コード     │ 必須   │ コード本文                        │
    └───────────┴────────┴─────────────────────────────────┘

  言語・タグの入力方法:
    - 入力欄をクリックすると既存の候補が一覧表示される
    - 文字を入力すると候補が絞り込まれる
    - 存在しない名前を入力してEnterを押すと新規作成できる
    - タグは複数選択でき、選択済みタグはチップ（バッジ）で表示される

  保存時の動作:
    - 保存後、登録・編集したスニペットが自動で選択状態になる
    - 新規登録した言語・タグは即時反映される
    - 「保存しました」の通知が出る

------------------------------------------------------------
■ スニペット削除
------------------------------------------------------------

  手順:
    1. 右ペインの「削除」ボタンをクリック
    2. 確認ダイアログが表示される
    3. 「削除」をクリックすると完全に削除される

  注意:
    - 削除後はデータを元に戻せない
    - 削除したスニペットのみで使われていたタグは自動で削除される

------------------------------------------------------------
■ データの保存場所
------------------------------------------------------------

  データベースファイル:
    dev-code-knowledge\data\snippets.db

  仕様:
    - SQLite形式のファイル1つにすべてのデータが保存される
    - アプリを終了してもデータは消えない
    - ファイルをコピーすればデータのバックアップになる
    - ファイルを削除するとデータがすべて消えるので注意
