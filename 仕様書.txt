■ 画面構成
------------------------------------------------------------

┌─────────────────────────────────────────────────────┐
│ [検索バー]                          [+ 新規登録]    │  ← 上部バー
├──────────┬──────────────────┬────────────────────────┤
│ 言語      │ スニペット一覧    │ コード詳細             │
│ タグ      │                  │                        │
└──────────┴──────────────────┴────────────────────────┘
  左ペイン    中央ペイン          右ペイン

------------------------------------------------------------
■ 検索機能
------------------------------------------------------------

【検索バー（上部）】

  検索対象:
    - タイトル
    - 説明
    - コード本文
    - タグ名

  仕様:
    - 文字を入力するたびにリアルタイムで絞り込まれる
    - 大文字・小文字を区別しない（例: "select" で "SELECT" もヒット）
    - 部分一致で検索される（例: "顧客" で "顧客一覧取得" がヒット）
    - 複数の検索対象を同時に見るため、タイトルになくてもコード内に
      あればヒットする
    - 入力欄の右端の × ボタンで検索をクリアできる

  例:
    "SELECT"  → コード本文に SELECT を含むスニペットがヒット
    "業務"    → タグ名が "業務系" のスニペットがヒット
    "顧客"    → タイトルが "顧客一覧取得" のスニペットがヒット

------------------------------------------------------------
■ 言語フィルター（左ペイン・上段）
------------------------------------------------------------

  仕様:
    - 登録済みの言語が一覧表示される
    - クリックするとその言語のスニペットだけに絞り込まれる
    - 選択中の言語はハイライト表示される
    - 「すべて」をクリックすると絞り込みが解除される
    - 言語の絞り込みと検索バーは同時に使える（AND条件）

  初期登録されている言語:
    SQL / C# / VBA / Python / JavaScript / PowerShell / その他

  注意:
    - 言語の追加・削除はアプリ上から直接はできない
    - 将来的に管理機能を追加予定

------------------------------------------------------------
■ タグフィルター（左ペイン・下段）
------------------------------------------------------------

  仕様:
    - スニペットに付けられたタグが自動で一覧に表示される
    - クリックするとそのタグを持つスニペットだけに絞り込まれる
    - 選択中のタグはハイライト表示される
    - 同じタグをもう一度クリックすると絞り込みが解除される
    - タグフィルター・言語フィルター・検索バーは同時に使える（AND条件）
    - タグが1つも登録されていない場合「タグなし」と表示される

  例:
    タグ「業務系」を選択 → 業務系タグが付いたスニペットだけ表示
    さらに言語「SQL」を選択 → 業務系 かつ SQL のスニペットだけ表示

------------------------------------------------------------
■ スニペット一覧（中央ペイン）
------------------------------------------------------------

  表示内容（1件につき）:
    - タイトル
    - 言語バッジ（青色）
    - タグバッジ（緑色）
    - 最終更新日

  並び順:
    - 更新日時の新しい順（降順）
    - 編集するたびに一番上に移動する

  操作:
    - クリックすると右ペインにコード詳細が表示される
    - 選択中のスニペットは左端に青い線が表示される

------------------------------------------------------------
■ コード詳細（右ペイン）
------------------------------------------------------------

  表示内容:
    - タイトル
    - 説明
    - 言語バッジ・タグバッジ
    - 最終更新日時（時刻まで表示）
    - コード本文（シンタックスハイライト付き）

  シンタックスハイライト対応言語:
    SQL / C# / VBA / Python / JavaScript / PowerShell

  操作ボタン:
    [コピー]  コード本文をクリップボードにコピーする
              → コピー成功時に「コピーしました！」と通知が出る
    [編集]    編集フォームを開く
    [削除]    確認ダイアログを表示してから削除する

  スニペット未選択時:
    「スニペットを選択してください」と表示される

------------------------------------------------------------
■ スニペット登録・編集フォーム
------------------------------------------------------------

  開き方:
    - 上部の「+ 新規登録」ボタン → 空の登録フォーム
    - 右ペインの「編集」ボタン   → 既存データが入力済みのフォーム

  入力項目:
    ┌───────────┬────────┬─────────────────────────────────┐
    │ 項目       │ 必須   │ 説明                             │
    ├───────────┼────────┼─────────────────────────────────┤
    │ タイトル   │ 必須   │ スニペットの名前                  │
    │ 説明       │ 任意   │ メモ・用途など                    │
    │ 言語       │ 必須   │ プルダウンから選択                │
    │ タグ       │ 任意   │ カンマ区切りで複数入力可能        │
    │ コード     │ 必須   │ コード本文（自動で縦に広がる）    │
    └───────────┴────────┴─────────────────────────────────┘

  タグの入力例:
    業務系, 月次, 集計
    → 「業務系」「月次」「集計」の3つのタグが付く

  タグの仕様:
    - 初めて入力したタグ名は自動で新規作成される
    - 既存のタグ名を入力すると既存のタグが再利用される
    - 前後のスペースは自動で除去される

  保存時の動作:
    - 保存後、登録・編集したスニペットが自動で選択状態になる
    - 新規登録したタグは左ペインのタグ一覧に即時反映される
    - 「保存しました」の通知が出る

------------------------------------------------------------
■ スニペット削除
------------------------------------------------------------

  手順:
    1. 右ペインの「削除」ボタンをクリック
    2. 確認ダイアログが表示される
    3. 「削除」をクリックすると完全に削除される

  注意:
    - 削除後はデータを元に戻せない
    - 削除したスニペットに紐づいていたタグは残る
      （他のスニペットが使っていなくても残る）

------------------------------------------------------------
■ データの保存場所
------------------------------------------------------------

  データベースファイル:
    dev-code-knowledge\data\snippets.db

  仕様:
    - SQLite形式のファイル1つにすべてのデータが保存される
    - アプリを終了してもデータは消えない
    - ファイルをコピーすればデータのバックアップになる
    - ファイルを削除するとデータがすべて消えるので注意

------------------------------------------------------------
■ 現在の制限事項・未実装機能
------------------------------------------------------------

  現在できないこと:
    - タグの複数同時選択（1つのみ）
    - 言語マスタの追加・編集（SQLiteを直接編集する必要あり）
    - 使われていないタグの削除
    - データのエクスポート・インポート
    - お気に入り登録
    - 使用履歴の記録

  将来追加予定:
    - 全文検索の高速化（SQLite FTS5）
    - AIによるコード説明
    - Markdown対応
    - エクスポート・インポート機能
