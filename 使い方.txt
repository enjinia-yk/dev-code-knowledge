============================================================
  CodeSnippet - 使い方ガイド
============================================================

■ 毎回の起動手順
------------------------------------------------------------

1. PowerShell（またはコマンドプロンプト）を開く

2. プロジェクトフォルダに移動する
   cd C:\path\to\dev-code-knowledge

3. 仮想環境を有効化する
   .venv\Scripts\activate

   → プロンプトの先頭に (.venv) が表示されれば成功
   例: (.venv) PS C:\path\to\dev-code-knowledge>

4. アプリを起動する
   python main.py

   → ブラウザが自動で開き http://localhost:8080 が表示される

------------------------------------------------------------
■ アプリの終了
------------------------------------------------------------

ブラウザを閉じた後、PowerShell で Ctrl + C を押す

------------------------------------------------------------
■ 仮想環境の無効化（作業終了時）
------------------------------------------------------------

deactivate

→ プロンプトから (.venv) が消えれば無効化完了

------------------------------------------------------------
■ VSCode を使っている場合（自動化する方法）
------------------------------------------------------------

1. VSCode でプロジェクトフォルダを開く

2. 右下のPythonバージョン表示をクリック
   （例: "Python 3.12.x" と表示されている箇所）

3. 一覧から以下を選択する
   .venv\Scripts\python.exe

→ 以後、VSCode のターミナルを開くたびに
  自動で仮想環境が有効化される

------------------------------------------------------------
■ よくあるトラブル
------------------------------------------------------------

【ブラウザが自動で開かない場合】
  手動で以下のURLをブラウザに入力する
  http://localhost:8080

【8080ポートが使用中と表示される場合】
  main.py の最終行のポート番号を変更する
  例: port=8081

【activate コマンドがエラーになる場合】
  PowerShell の実行ポリシーが制限されている可能性がある
  以下のコマンドを管理者権限で実行する
  Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
