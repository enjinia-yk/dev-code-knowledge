============================================================
  CodeSnippet - 使い方ガイド
============================================================

■ 毎回の起動手順
------------------------------------------------------------

1. PowerShell（またはコマンドプロンプト）を開く

2. プロジェクトフォルダに移動する
   cd C:\path\to\dev-code-knowledge

3. 仮想環境を有効化する
   .venv\Scripts\activate

   → プロンプトの先頭に (.venv) が表示されれば成功
   例: (.venv) PS C:\path\to\dev-code-knowledge>

4. アプリを起動する
   python main.py

   → ブラウザが自動で開き http://localhost:8080 が表示される

------------------------------------------------------------
■ アプリの終了
------------------------------------------------------------

ブラウザを閉じた後、PowerShell で Ctrl + C を押す

------------------------------------------------------------
■ 仮想環境の無効化（作業終了時）
------------------------------------------------------------

deactivate

→ プロンプトから (.venv) が消えれば無効化完了

------------------------------------------------------------
■ VSCode を使っている場合（自動化する方法）
------------------------------------------------------------

1. VSCode でプロジェクトフォルダを開く

2. 右下のPythonバージョン表示をクリック
   （例: "Python 3.12.x" と表示されている箇所）

3. 一覧から以下を選択する
   .venv\Scripts\python.exe

→ 以後、VSCode のターミナルを開くたびに
  自動で仮想環境が有効化される

------------------------------------------------------------
■ 検索の使い方
------------------------------------------------------------

検索欄にキーワードを入力するとリアルタイムで絞り込まれる

【検索対象】
  タイトル・説明・コード・言語名・タグ名

【AND 検索】
  スペースで区切ると複数ワードの AND 検索になる
  例: "SQL 顧客" → 言語がSQLかつ「顧客」を含むスニペット

【大文字・小文字】
  区別しない（"sql" で "SQL" もヒットする）

------------------------------------------------------------
■ PC 起動時に自動で起動する設定
------------------------------------------------------------

start_hidden.vbs をスタートアップフォルダに配置すると
PC 起動時にアプリが自動で起動する（ターミナルは表示されない）

1. スタートアップフォルダを開く
   Win + R → shell:startup と入力して Enter

2. start_hidden.vbs のショートカット（または本体）を
   スタートアップフォルダにコピーする

※ start_hidden.vbs の batPath の行は環境に合わせて書き換えること
  - 環境変数版: %USERPROFILE%\workspace\dev-code-knowledge\start.bat
  - 絶対パス版: start_hidden_absolute.vbs を参考に実際のパスを記入する

------------------------------------------------------------
■ 自動起動版の終了方法
------------------------------------------------------------

ターミナルが表示されないため Ctrl + C では終了できない

1. タスクマネージャーを開く（Ctrl + Shift + Esc）
2. 「詳細」タブをクリック
3. python.exe を右クリック →「タスクの終了」

------------------------------------------------------------
■ 他のPCからアクセスする（同一LAN内）
------------------------------------------------------------

アプリを起動しているPCのIPアドレスを確認する
  ipconfig → IPv4 アドレスをメモ（例: 192.168.1.10）

他のPCのブラウザで以下を開く
  http://192.168.1.10:8080

※ 初回のみ Windowsファイアウォールでポート8080の許可が必要
  管理者権限の PowerShell で以下を実行する
  New-NetFirewallRule -DisplayName "CodeSnippet" -Direction Inbound -Protocol TCP -LocalPort 8080 -Action Allow

------------------------------------------------------------
■ 他のPCで環境を構築する手順
------------------------------------------------------------

1. リポジトリをクローンする
   git clone https://github.com/enjinia-yk/dev-code-knowledge.git
   cd dev-code-knowledge

2. 仮想環境を作成・有効化する
   python -m venv .venv
   .venv\Scripts\activate

3. 依存パッケージをインストールする
   pip install -r requirements.txt

4. アプリを起動する
   python main.py

------------------------------------------------------------
■ よくあるトラブル
------------------------------------------------------------

【ブラウザが自動で開かない場合】
  手動で以下のURLをブラウザに入力する
  http://localhost:8080

【8080ポートが使用中と表示される場合】
  main.py の最終行のポート番号を変更する
  例: port=8081

【activate コマンドがエラーになる場合】
  PowerShell の実行ポリシーが制限されている可能性がある
  以下のコマンドを管理者権限で実行する
  Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
